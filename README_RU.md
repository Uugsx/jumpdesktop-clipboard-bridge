# Мост для синхронизации изображений через буфер обмена в Jump Desktop

[Read in English (English Version)](README.md)

Этот проект предлагает простое и быстрое решение для прямого копирования и вставки скриншотов и изображений между macOS и удаленной сессией Windows в **Jump Desktop**.

## Проблема
Jump Desktop не поддерживает прямую синхронизацию буфера обмена для изображений (например, копирование скриншота на Mac и вставка его в Windows). Он синхронизирует только текстовые записи.

## Решение
Этот мост обходит данное ограничение, используя быструю передачу текста в Jump Desktop:
1. **На macOS**: Фоновый процесс следит за буфером обмена. Когда вы копируете изображение (например, делаете скриншот), он автоматически преобразует его в Base64-строку и кладет в буфер обмена и как изображение (для приложений macOS), и как текст (для RDP-сессии).
2. **Jump Desktop**: Мгновенно переносит текстовый буфер на удаленную машину Windows.
3. **На Windows**: Легковесный скрипт PowerShell следит за буфером обмена Windows. Когда он видит закодированный текст скриншота, он раскодирует его обратно в изображение, стирая текст.

Теперь вы можете нажать `Ctrl + V` в Windows (Outlook, Word, Проводник, Telegram), и картинка вставится мгновенно!

## Установка и настройка

### 1. Настройка macOS (Локальный компьютер)
Запустите следующие команды в Терминале на вашем Маке:

```bash
# Перейдите в папку с файлами
cd jumpdesktop-clipboard-bridge

# Разрешите запуск скрипта установки и выполните его
chmod +x install.sh
./install.sh
```

Этот скрипт скомпилирует службу на Swift, сохранит её в папку `~/.jump_clipboard_bridge/` и добавит в автозагрузку macOS (LaunchAgents), чтобы всё работало автоматически при входе в систему.

### 2. Настройка Windows (Удаленный компьютер)
Права администратора на удаленной Windows **не требуются**.

1. Скопируйте файл `sync_clipboard.ps1` на удаленный компьютер (например, на Рабочий стол).
2. **Как запустить**: Кликните правой кнопкой мыши по файлу `sync_clipboard.ps1` и выберите **Выполнить с помощью PowerShell** (Run with PowerShell).
3. Оставьте синее окно PowerShell свернутым на заднем плане.

*Совет: Если вы случайно кликнули мышкой внутри синего окна PowerShell и его заголовок изменился на «Выбрать» (или «Select»), выполнение скрипта приостановится. Нажмите `Enter` или `Esc` в этом окне, чтобы возобновить работу. Чтобы отключить эту паузу навсегда, кликните правой кнопкой по заголовку окна -> Свойства -> снимите галочку «Выделение мышью» (QuickEdit Mode).*

---

## Как пользоваться
1. Сделайте скриншот на Маке (через стандартные клавиши macOS или Flameshot).
2. Перейдите в окно удаленного стола Windows.
3. Нажмите **`Ctrl + V`** в любом приложении (Outlook, Word, проводник или чат). Изображение будет вставлено моментально.

---

## Удаление (macOS)
Чтобы остановить службу и полностью удалить файлы с Мака:
```bash
launchctl unload ~/Library/LaunchAgents/com.jump.clipboard-bridge.plist
rm -f ~/Library/LaunchAgents/com.jump.clipboard-bridge.plist
rm -rf ~/.jump_clipboard_bridge
```
